# FPGA File Support

**fpga-file-support** provides syntax highlighting for FPGA-related files:

- **Verilog / SystemVerilog** (`.v`, `.sv`, `.vh`, `.svh`)
- **TCL scripts** (`.tcl`)
- **Constraint files**:  
  - SDC (`.sdc`)  
  - XDC (`.xdc`)  
  - UCF (`.ucf`)  
  - CST (`.cst`)

## ✨ Features

✅ Syntax highlighting for all major FPGA design files  
✅ Works seamlessly with other HDL extensions
